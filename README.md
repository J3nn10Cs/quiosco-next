# 🚀 Quiosco con Next.Js

- NextJs, Prisna
- **Otras Herramientas:** JavaScript, ESLint, Tailwind

## 📥 Instalación
### 1️⃣ Configurar Variables de Entorno
```env
  DATABASE_URL
```

### 2️⃣ Instalar Dependencias
Ejecuta los siguientes comandos

```sh
  npm install
```

## ▶️ Ejecución
Para correr el proyecto, abre dos terminales y ejecuta:

```sh
npm run dev
```

```sh
npx prisma studio
```
